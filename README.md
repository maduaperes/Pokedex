# Pokédex | Buscar Pokémon

Simples e moderno para buscar informações sobre Pokémon por nome ou ID, com visual inspirado no universo Pokémon.

---

## Descrição

Este projeto é uma Pokédex online que permite ao usuário digitar o nome ou ID de um Pokémon e obter informações básicas, como imagem, nome, tipos e outras características. O design é totalmente inspirado na estética Pokémon, incluindo uma pokébola estilizada para exibir os resultados.

---

## Tecnologias utilizadas

- HTML5
- CSS3 (com animações e tema Pokémon)
- JavaScript (fetch API para buscar dados da PokéAPI)

---

## Funcionalidades

- Busca de Pokémon pelo nome ou ID
- Exibição estilizada dos dados dentro de uma Pokébola animada
- Feedback visual elegante e responsivo
- Fonte pixelada inspirada em jogos clássicos Pokémon
